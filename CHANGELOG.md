# Changelog

> **约定**（2026-07-12 起）：每次发版（打 tag）后在这里给该版本写一段**人话说明**（用户视角优先，开发侧改动一笔带过），
> 并把同一段同步到对应 GitHub Release 页的顶部（Release 页保留自动生成的 PR 列表给开发者看细节）。
> 正本是本文件；历史版本（v0.0.1–v0.6.6）为 2026-07-12 回溯补写。
> 版本号：tag 即版本真源（`git tag vX.Y.Z && git push origin vX.Y.Z` 触发签名发版），带新功能 bump 0.1，纯修复 bump patch。

## v0.8.0 — 2026-07-12

**文档互链和分页排版进正式版。**

- **文档互链**：文档里可以链接到另一篇文档——`@` 提及快速插入、悬停气泡预览、从侧栏拖文件进正文成链接；改名/移动后链接自动跟随
- **分页文档**：Word 式分页排版——统一 A4 页高、页面设置、分页线、导出 PDF 带页码；表格/代码块跨页正确分割

## v0.7.0 — 2026-07-12

**Wordspace 现在也是一个浏览器。** 网页和文档共用同一套侧栏、地址栏和标签系统。

- 地址栏上网（输网址或直接搜索，默认 Bing）+ 自动补全（开着的标签 / 收藏 / 历史）
- 收藏夹：地址栏 ☆、侧栏折叠收藏区、收藏管理页；和 Chrome / Safari / Firefox / Edge 互导书签
- 历史记录（按日期分组、可搜索、按时段清除）、起始页（搜索框 + 收藏瓦片）
- 后退/前进/刷新、页内查找（⌘F）、缩放（⌘±0）、原生右键菜单、会话恢复、全套快捷键（⌘T/⌘W/⌘⇧T/Ctrl+Tab/⌘1-9）
- 网页标签与文档标签混装、置顶、拖拽。定位「够用的标准浏览器」：不做下载/扩展/剪藏
- 修复：新标签加载中短暂闪回文档；一轮多角度审查修掉一批边角问题（含几个丢未保存内容的情况）

## v0.6.6 — 2026-07-11

- **打开桌面/大文件夹不再卡**：文件树跳过 macOS 应用包（.app 等）和依赖目录（node_modules 等），不再把包内几万个文件当文件树读
- 开发侧：跨 session 对齐体系（team-memory 公告板 + feature 对齐 spec）；浏览器完整规格定稿（为 v0.7.0 铺路）；ui-demo 分页文档完整版

## v0.6.5 — 2026-07-10

- **性能诊断模式**（菜单「Wordspace Next → 性能诊断…」）：卡顿时可自查每个文件夹的读树耗时/渲染耗时/主线程长任务，一键录 CPU Profile 发给我们排查；普通使用零感知
- 开发侧：ui-demo 浏览器功能全套 + 文档区 back/forward 原型（为 v0.7.0/后续铺路）

## v0.6.4 — 2026-07-09

- **修复一个数据安全问题**：文档里点相对链接后，自动保存可能把目标页内容写进原文件——现已收口，文档内链接点击统一走应用内打开
- 关闭标签后激活相邻标签（原来会跳到列表最后一个），文件树不再跟着乱滚
- 多文件夹工作区的渲染性能优化（增量渲染，展开/折叠不再整树重建）

## v0.6.3 — 2026-07-08

- **多文件夹工作区**：可以同时打开多个文件夹当工作区，侧栏并排显示；智能处理嵌套（子文件夹并入父文件夹）；文件夹失联（如外置盘拔出）显示灰态可重新定位
- **跨文件夹移动文件**：文件树里直接把文件拖到另一个文件夹的树里
- 开发侧：ui-demo 文档互链/分页原型（为 v0.8.0 铺路）

## v0.6.2 — 2026-07-06（发版流水线中断，无安装包；改动随 v0.6.3 到达用户）

- 「纸方墨圆」设计语言移植进 app（保守阴影口径）；文件树深层嵌套改 compact folders（单子链合并成一行）+ 淡墨导引线；滚动时祖先文件夹吸顶；侧栏滚动条跳动修复

## v0.6.1 — 2026-07-05

- **全面 bug sweep**：多路对抗审计修掉约 40 个问题，含两个严重项（删除线操作掉出块编辑器、跨块删除损坏表格）和一批丢数据边角

## v0.6.0 — 2026-07-03

- **Markdown 支持**：.md 文件可以直接打开、编辑、保存，和 HTML 文档用同一个块编辑器
- **AI 接入入口**：菜单/侧栏左下角「AI 接入」——复制 Prompt 或安装 Skill，让你的 AI（Claude Code/Cursor 等）会写 Wordspace 文档
- macOS 关窗改为「隐藏驻留」：红叉/⌘W 关窗后 app 留在 Dock，再点秒开（标签/未保存内容都在）
- 从 Finder 双击打开文件时，侧栏文件树自动展开定位到它

## v0.5.0 — 2026-07-02

- **Schema 地基**（编辑器架构升级）：打开文件时自动校验——符合 Wordspace 格式走完整块编辑，不符合（野生 HTML）自动降级为基础文字编辑并提示，不再出现「编辑坏文件损坏内容」
- **临时文档保存流**：新建的文档是未保存的临时文档，保存时再选存到哪个文件夹（带保存对话框、未保存关闭确认）
- 侧栏真收起（⌘\）+ ⌘P 快速打开（模糊搜文件名）
- 导出收敛为单按钮 PDF 导出（所见即所得）；另存为、右上角 ⋯ 菜单
- 一大批与 ui-demo 的视觉/交互对齐修复
