把公司常用文档（手册、标书、演示稿等）的版式存成模板，之后人或 Agent 都能调用模板快速生成一份带公司既有样式的文档。两类来源：公司内部私有模板库 + 公开共享模板池（允许跨组织复用，Agent 也能取用）。降低「从空白开始」的成本，让团队文档风格保持一致。
