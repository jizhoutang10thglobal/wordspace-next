打开本地 .html 文件，在所见即所得的块编辑器里直接改：选中文字弹浮动工具栏（粗体/斜体/颜色/高亮），斜杠菜单插入标题/列表/分隔线，拖拽手柄调块顺序。保存时剥掉编辑器自身的标记、把文档逐字写回原 HTML 文件，保真不破坏模板。表格、图片等复杂块锁定保留、可整块删除但不误改。这是 Wordspace「人手做轻量编辑」那一层的落地，已随 v0.1.x ship（mac 签名公证 + Windows 包）。
