## 这是什么

电脑里有一种文件叫 HTML 文件——网页，还有公司很多正式文档（价值观、手册、招聘页）其实都是这种文件。麻烦在于：想改它，以前得会写代码。

这个功能让你**不用懂代码，直接改**。打开一个 HTML 文件，你看到的就是它本来排好版的样子（像一张做好的纸，不是一堆代码）。你点一下文字就能改，跟在 Word 或 Notion 里打字一样。改完一保存，文件还是原来那个 HTML 文件、只是内容变了——你没动过的地方，一个字都不会乱。

## 能干什么

- 打开电脑里的一个 .html 文件，看到它排好版的样子。
- 点进去**直接改文字**：打字、删字、换内容。
- **简单排版**：加粗、斜体、改颜色、高亮；做标题、做列表。
- **整块搬动**：一段话、一个标题，能拖着上下挪位置。
- **插入东西**：打一个「/」弹出菜单，插入标题、列表、分隔线之类。
- **保存**：写回原文件，保真——你没碰的部分（样式、表格、图片、脚本）原样保留，不被搞坏。
- 撤销 / 重做；粘贴时只粘纯文字（不把别处的乱格式带进来）。

## 现在做到哪了

基础的已经能用：块编辑、浮动工具栏、斜杠菜单、拖块、保存写回、历史雏形（都随 v0.1.x 出过了）。**但还很多没做**——更多排版能力、更多种类的块、更顺手的交互。具体差什么、抄哪些，参考 heyhtml.com 和 github 上的 cxq0517/htmltool2。这块是第一阶段的大头，会一直打磨。

## 算做完的标准

打开一份真实的公司文档（带标题、列表、表格、配图），改几处文字 + 改点格式，保存，再打开：改的地方在、没改的地方一字没乱、整个文档没被搞坏。

## 范围

只管**本地文件**（电脑上的文件）。多人协作、云端这些是第二阶段的事，这里不碰。
