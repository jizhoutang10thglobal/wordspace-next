# wordspace.ai — robots policy
# wordspace is a local HTML document editor and wants to be
# discoverable by both traditional search engines and AI answer engines.
# The default is "allow everyone"; any future blocks should require an
# explicit product-strategy sign-off.

# Default allow for all crawlers (covers Googlebot, Bingbot, DuckDuckBot,
# Yandex, and anything else respecting robots.txt).
User-agent: *
Allow: /
Disallow: /downloads/mac
Disallow: /downloads/win

# Explicit opt-ins for AI crawlers. Listing each UA explicitly (rather than
# relying on the wildcard above) means future UA-specific bans or rate
# limits have an obvious place to land without surprise behavior change.
